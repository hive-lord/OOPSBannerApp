# OOPSBannerApp

Repository for my college **STEP CLASS** work.

This repo contains:
- **Use Cases**: the tasks given in class
- **Weekly Assignments**: assignments given each week

## Language
- **Java** (100%)

## How to Use / Run

### Prerequisites
- Java JDK (11+ recommended)
- Any IDE (IntelliJ / Eclipse / VS Code) or command line

### Run (general)
1. Clone the repo:
   ```bash
   git clone https://github.com/hive-lord/OOPSBannerApp.git
   cd OOPSBannerApp
   ```

2. Open in your IDE and run the class that contains `public static void main(String[] args)`.

### Run (CLI example)
> Adjust the main class name/package to match your project.

```bash
javac -d out $(find . -name "*.java")
java -cp out Main
```

## Folder/Work Organization

Typical structure (may differ in this repo):
- `UseCases/` (or similar): class tasks / use-cases
- `Weekly-Assignment/` (or similar): weekly assignment submissions
- `src/`: Java source files (if used)

## Notes
- Each program may be separate and runnable on its own.
- If multiple `main` methods exist, run the one for the specific task/assignment you want.
